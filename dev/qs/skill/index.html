提示

通过查看spring boot 和 spring cloud的版本关系来确定他们之间的版本。查看官网版本对应明细 (opens new window)。 通过查看spring cloud alibaba系列产品组件之间的版本Wiki (opens new window)，来确定和spring cloud版本之间的关系。
