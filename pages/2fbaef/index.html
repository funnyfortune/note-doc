流程表单

# 功能描述

说明

整合element ui动态表单，由于大部分后台需求都是居于表单的开发，方便前端开发人员通过界面化进行表单的布局和开发，也支持提供流程表单的动态创建与绑定，最大限度保证表单设计的灵活性。 Element UI表单设计及代码生成器，可将生成的代码直接运行在基于Element的vue项目中；也可导出JSON表单，使用配套的解析器将JSON解析成真实的表单。 在线配置动态表单，无需建表与开发，通过图形界面进行表单设计。

# 操作步骤

# 1.列表

列表包含了[新增，修改，删除，详情]功能按钮，具有相关权限则可见，可使用。

打开 [工作流程->流程管理->流程表单] 菜单，可以看到对应的界面，如下图所示：

# 2.新增/修改

点击[新增]或[修改]按钮进入到新增或编辑页面，设计相应的流程表单，如下图所示：

# 3.删除

在列表界面，选中相应的条目之后，点击右侧的[删除]按钮将会出现删除确认信息弹框，选中后点击”确定”按钮即可进行删除操作。
