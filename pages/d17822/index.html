导入导出

组件 myapp-common-poi，基于 EasyExcel 实现 Excel 的读写操作，可用于实现最常见的 Excel 导入导出等功能。未来将扩展该组件支持 pdf， excel， word, ppt 等文件在线功能。

EasyExcel 介绍

EasyExcel (opens new window) 是阿里开源的 Excel 工具库，具有简单易用、低内存、高性能的特点。

easyexce l重写了poi对07版Excel的解析，一个3M的excel用POI sax解析依然需要100M左右内存，改用easyexcel可以降低到几M，并且再大的excel也不会出现内存溢出。

# @ExcelProperty

用于设置单元格属性的注解。参数如下：

参数 含义
value 字段名称，如果指定了多个值，会自动进行合并
index 属性在Excel中列的下标，列的绝对位置，下标从0开始
order 列的排序优先级，若不指定，则按所在类的位置排序，order的默认值为Integer.MAX_VALUE，order值越小，越排在前面，优先级：index > order > 默认配置
converter 数据类型转换，EasyExcel 内置了大量转换器，请查看 com.alibaba.excel.converters，避免重复造轮子

# 创建导出 API

    /**
     * 导出字典类型列表
     */
    @ApiOperation(value = "导出")
    @ApiImplicitParams({
            @ApiImplicitParam(name = "Authorities", value = "用户token", required = true, dataTypeClass = String.class, paramType = "header")})
    @OperateLog(ignoreResponseData = true)
    @PreAuthorize("@auth.hasPerm('dict:export')")
    @GetMapping("/export")
    public void export(SysDictTypeListDTO sysDictType, HttpServletResponse response) throws IOException {
        List<SysDictTypeListVO> list = sysDictTypeService.selectDictTypeList(sysDictType);
        EasyExcel.write(getFileOutputStream("字典类型.xlsx", response), SysDictTypeListVO.class).sheet("数据").doWrite(list);
    }

# 转换器

在读写 Excel 时，有时候需要我们进行数据类型转换，例如字典类型，时间戳，在实体对象中是 Long 或 int 类型，但是这样直接导出到 Excel 中不太直观。我们需要转换成 字典名称 或 yyyy-MM-dd HH:mm:ss 格式，此时我们就可以用到转换器。

EasyExcel 定义了 Converter (opens new window) 接口，用于实现字段的转换。它有两个核心方法：

① #convertToJavaData(...) 方法：将 Excel Row 对应表格的值，转换成 Java 内存中的值。例如说，Excel 的“状态”列，将“状态”列转换成 status = 1 ，”禁用”列转换成 status = 0 。

② #convertToExcelData(...) 方法：恰好相反，将 Java 内存中的值，转换成 Excel Row 对应表格的值。例如说，Excel 的“状态”列，将 status = 1 转换成“开启”列， status = 0 转换成”禁用”列。

# 字典转换器

  1. DataDictConverter 数据字典转换器需要配合 @DictFormat 注解指定数据字典类型来使用。使用方式如下：
   @DictFormat(DictTypeEnum.SYS_COMMON_STATUS)
   @ExcelProperty(value = "合同状态",converter = DataDictConverter.class)
   private int status;

  1. DictStringConverter 数据字典转换器需要配合 @NumberFormat 注解指定字典数据来使用。使用方式如下：
  @NumberFormat("0=新签&1=续签&2=变更&3=终止&4=解除")
  @ExcelProperty(value = "合同状态",converter = DictStringConverter.class)

# @ColumnWith

用于设置列宽度的注解，注解中只有一个参数 value 。 value 的单位是字符长度，最大可以设置 255 个字符，因为一个 Excel 单元格最大可以写入的字符个数，就是 255 个字符。

最佳实践

public class ImeiEncrypt {
   
   @ColumnWidth(value = 18)
   private String imei;
}

# @ContentFontStyle

用于设置单元格内容字体格式的注解。参数如下：

参数 含义
fontName 字体名称

italic 是否斜体
strikeout 是否设置删除水平线
color 字体颜色

# @ContentStyle

设置内容格式注解。参数如下：

参数 含义
dataFormat 日期格式
hidden 设置单元格使用此样式隐藏
locked 设置单元格使用此样式锁定
quotePrefix 在单元格前面增加`符号，数字或公式将以字符串形式展示
horizontalAlignment 设置是否水平居中
wrapped 设置文本是否应换行。将此标志设置为true通过在多行上显示使单元格中的所有内容可见
verticalAlignment 设置是否垂直居中
rotation 设置单元格中文本旋转角度。03版本的Excel旋转角度区间为-90°~90°，07版本的Excel旋转角度区间为0°~180°
indent 设置单元格中缩进文本的空格数
borderLeft 设置左边框的样式
borderRight 设置右边框样式
borderTop 设置上边框样式
borderBottom 设置下边框样式
leftBorderColor 设置左边框颜色
rightBorderColor 设置右边框颜色
topBorderColor 设置上边框颜色
bottomBorderColor 设置下边框颜色
fillPatternType 设置填充类型
fillBackgroundColor 设置背景色
fillForegroundColor 设置前景色
shrinkToFit 设置自动单元格自动大小
