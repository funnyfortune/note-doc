基础数据

# 功能描述

说明

基础数据的功能是对人事管理模块的基础数据进行可动态化配置和维护，也对一些标准数据进行预先录入，不用用户自建一些繁琐的基本数据。

基础数据包含了11个菜单功能：[离职原因]、[合同类型]、[员工类型]、[国籍]、[民族]、[学历]、[血型]、[政治面貌]、[变更原因]、[户口性质]、[证件类型]

基础数据包含了系统内置和用户自定义两种类型。系统内置是不可以变更，属于系统预设，全租户可见；用户自定义则是可以修改删除，具有租户隔离性。

# 操作步骤

# 1.列表

列表包含了[新增，修改，删除，导出]功能按钮，具有相关权限则可见，可使用。

打开 [人事管理-基础配置] 菜单，可以看到对应的界面，如下图所示：

# 2.新增/修改

点击[新增]或[修改]按钮进入到新增或编辑页面，填写相关表单信息,其中带*号信息为必填信息，如下图所示：

# 3.删除

在列表界面，选中相应的条目之后，点击右侧的[删除]按钮将会出现删除确认信息弹框，选中后点击”确定”按钮即可进行删除操作。

# 4.导出

在列表界面，根据查询条件，导出相应excel文件。
