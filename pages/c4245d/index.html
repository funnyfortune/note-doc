# 功能描述

说明

在线用户居于租户级别可视化在线用户的详细情况。

# 操作步骤

# 1.列表

列表包含了[强退]功能按钮，具有相关权限则可见，可使用。 打开 [运维监控-> 在线用户] 菜单，可以看到对应的列表，如下图所示：

# 2.强退

在列表界面，选中相应的用户之后，点击右侧的”强退”按钮将会出现确认强制下线该用户弹框，选中后点击”确定”按钮即可进行用户强行退出行为。
