组织部门

功能描述

在企业开发中，当企业发展超过一定的规模时，或者分公司的模式，分公司的人员都由自己管理，这时就需要分级管理，分级管理员可以管理授权给他的部门及这个部门下的用户，同时分级管理员可以再次分级授权。

# 操作步骤

用户管理列表界面拥有[新增，修改，导出，删除，同步]按钮，需要具有该按钮的权限则可使用。 点击 [组织管理-->部门管理] 菜单，查看列表。如下图所示：

# 1.新增/编辑部门

点击[新增]或[修改]按钮进入到新增或编辑页面，填写相关表单信息,其中带*号信息为必填信息，如下图所示：

部门管理必须先创建，否则无法添加人员。顶级组织，负责人是不需要填写，非顶级部门，必须指定负责人，用于流程的审核与消息操作。

组织类型是为了之后作为数据权限的一个重要依据，根据不同的组织类型，用户拥有不同的权限，为了达到一个用户可以夸分公司或子公司的组织类型进行查看数据（待实现）

# 2.删除

删除操作必须确保该部门的子部门已删除和该部门不再绑定用户，否则删除失败。

# 3.同步

若信息被新增修改删除，希望马上生效，则可以通过手动进行同步，若不同步，则会由任务调度中心定时调度更新缓存信息。
