# 如何注册

Vue 注册组件有两种方式：全局注册、局部注册。

# 局部注册

在对应的 Vue 页面中，使用 components 属性来注册组件。代码如下：

<template>
 <countTo :startVal='startVal' :endVal='endVal' :duration='3000'></countTo>
</template>

<script>
import countTo from 'vue-count-to';
export default {
 components: { countTo }, // components 属性
 data () {
   return {
     startVal: 0,
     endVal: 2017
   }
 }
}
</script>

# 组件通信

# 通过 Prop 向子组件传递数据

基于上述的 Button-counter 示例组件，讲解父子组件如何通信。 通过 props 定义一个initCount 来接受父组件传过来的初始值。代码如下：

<!-- 子组件 -->
<template>
 <div><button v-on:click="count++">You clicked me {{ count }} times.</button></div>
</template>
<script>
export default {
 name: "Button-counter",
 props: {
   initCount: Number
 },
 data: function () {
   return {
     count: this.initCount
   }
 },
</script>

<!-- 父组件 -->
<template>
 <div id="components-demo">
   <button-counter :init-count="1"></button-counter> <!-- 2. :name 传入 -->
 </div>
</template>
<script>
import buttonCounter from "@/components/button-counter"; // 1. 引入
export default {
 components: { buttonCounter } // 2. 注册
};
</script>

一个组件默认可以拥有任意数量的prop，任何值都可以传递给任何 prop。在上述模板中，你会发现我们能够在组件实例中访问这个值，就像访问data 中的值一样。

# 监听子组件事件

子组件可以通过调用内建的$emit 方法并传入事件名称来触发一个事件, 不同于组件和prop，事件名不会被用作一个 JavaScript 变量名或 property 名，所以就没有理由使用 camelCase 或 PascalCase 了。

并且 v-on 事件监听器在 DOM 模板中会被自动转换为全小写 (因为 HTML 是大小写不敏感的)，所以 v-on:myEvent 将会变成v-on:myevent——导致myEvent 不可能被监听到。 因此，我们推荐你始终使用kebab-case 的事件名。

this.$emit('myEvent')
1

子组件通过 $emit 方法，让父组件监听到自定义事件。代码如下：

<!-- 子组件 -->
<template>
 <div><button v-on:click="myClick">You clicked me {{ count }} times.</button></div>
</template>
<script>
export default {
 name: "Button-counter",
 props: {
   initCount: Number
 },
 data: function () {
   return {
     count: this.initCount
   }
 },
 methods: {
   myClick() {
     this.count++;
       // 向父组件传递一个change事件，并把更新后的值传过去
     this.$emit('change', this.count);
   }   
 }
}
</script>

<!-- 父组件 -->
<template>
 <div id="components-demo">
   <button-counter @change="changeEvt"></button-counter> <!-- 2. :name 传入 -->
 </div>
</template>

<script>
import buttonCounter from "@/components/button-counter"; // 1. 引入
export default {
 components: { buttonCounter }, // 2. 注册
 props: {
   initCount: Number
 },
 data: function () {
   return {
     count: this.initCount
   }
 },
 methods: {
   changeEvt(count) {
    console.log(count); //接收子组件更新的值
   }   
 }
}
</script>
